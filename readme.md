# DEVCAMPER API

> Backend API for DevCamper Application, which is a bootcamp directory website

## Usage

Rename "config/config.env.env" to "config/config.env" and update values / settings

## Install dependencies

`npm install`

## Run App

```
#run in dev mode
npm run dev

#run in production mode
npm start
```
